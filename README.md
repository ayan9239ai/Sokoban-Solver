# Terminal Sokoban Solver & Player

A Python-based implementation of the classic puzzle game **Sokoban**, featuring an automated solver using Breadth-First Search (BFS).

## 🚀 Features
* **Manual Play:** Use `W`, `A`, `S`, `D` to navigate and push boxes. Finally It will print your solution.
* **Auto-Solver:** Finds the shortest LURD (Left, Up, Right, Down) path automatically.

## 📥 Installation
1. Ensure you have **Python 3.x** installed.
2. Clone or download this directory.
3. No external libraries are required (uses built-in `os`, `sys`, and `collections`).

## 🎮 How to Run
Run the script to enjoy both automatic and manual solver SOKOBAN via your terminal:
```bash
python Sokoban(Task2).py
